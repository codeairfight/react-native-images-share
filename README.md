
```js
import ShareImages from 'react-native-images-share';
```

Simple as calling:
```js
ShareImages.show({
  images: [''], // list images base64
  exclude: ['']
});
```
#### Note:
`images` is list base64 of images. Example: ['3hG2424HYu23...', 'Hy234hhj234g...', ...]
